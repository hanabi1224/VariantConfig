## Demo website

### Install

+ [yarn](https://classic.yarnpkg.com/en/docs/install/#windows-stable)
+ [node](https://nodejs.org/en/download/)

Run

```
./run.sh
```

```
run.bat
```

### Build web pages

```
cd pages
yarn
yarn build
```

### Run server

```
cargo run
```

Open [http://localhost:8080/](http://localhost:8080/)
